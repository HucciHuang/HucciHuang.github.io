`*` 和 `**` 在定义函数、调用函数、序列解包时，分别有着不同的用法。

## 定义函数

定义函数时，如果在参数前加上 `*` ，意味着可以传入多个非关键词参数；如果加上 `**` ，意味着可以传入多个关键词参数。

```python
def foo1(*args):
		pass
def foo2(**kwargs):
		pass

# 传入多个非关键词参数
foo1(1,2,3)
# 传入多个关键词参数
foo2(a=1, b=2, c=3)
```

## 调用函数

调用函数时可以对序列解引用。

```python
long_string = 'hello,world'
print(*long_string.split(','))
```

## 序列解包

序列解包的时候，可以用一个变量名定义序列。

```python
a, *b = [1,2,3,4]
print(a) # 1
print(b) # [2,3,4]
```
